# ERNOTE BY ERADA PLUS CO.LLC

تطبيق تجربة قابل للرفع والتعديل مبني بـ React + Vite.

## التشغيل على جهازك

```bash
npm install
npm run dev
```

ثم افتح الرابط الذي يظهر لك.

## الرفع على Vercel

1. ارفع هذا المجلد على GitHub.
2. افتح Vercel.
3. اختر Add New Project.
4. اختر الريبو.
5. اضغط Deploy.

## التعديل

- ملف التطبيق الرئيسي: `src/main.jsx`
- ملف التصميم والألوان: `src/styles.css`
- الشعار: `public/erada-logo.png`

## ملاحظة

هذه نسخة Frontend تجريبية تحفظ البيانات في LocalStorage.
لجعلها نظامًا حقيقيًا متعدد المستخدمين، اربطها لاحقًا مع Supabase أو Firebase.
